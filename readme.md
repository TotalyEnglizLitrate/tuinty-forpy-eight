## Installation

1. Clone the repository:
    ```
    git clone https://github.com/TotalyEnglizLitrate/tuinty-forpy-eight
    ```

2. Install textual
    ```
    pip install textual=0.41.0
    ```

3. Infinite fun, at your terminal!
